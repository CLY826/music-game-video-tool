## 项目简介

本项目面向音游玩家，开发一款 Android 端视频练习辅助应用。用户可按游戏名称对视频进行分类管理，支持视频变速播放、片段标记与 AB 循环播放，方便反复练习难点操作。同时结合后端服务与云存储，实现视频上传、列表查询与下载功能，满足本地处理与云端同步需求。整体设计简洁实用，助力玩家高效练习音游难点。

## 技术栈(初步规划)

前端: Kotlin
后端: SpringBoot
数据库: MySQL
