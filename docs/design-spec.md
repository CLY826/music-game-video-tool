## 三、核心页面与功能说明

### 1. 登录页

- **功能**：提供用户登录入口，支持「邮箱/手机号 + 密码」验证。
- **入口**：新用户可点击「新用户注册」跳转至注册页。
- **状态**：登录成功后进入游戏列表页（首页）；登录失败提示错误信息。

### 2. 注册页

- **功能**：新用户账号创建，包含用户名、手机号、邮箱、密码、确认密码输入项。
- **校验**：密码与确认密码需一致，手机号/邮箱格式需合法。
- **跳转**：注册成功后自动跳转至登录页。

### 3. 游戏列表页（首页）

- **功能**：展示用户可选择的音游列表（如 Phigros、Project Sekai 等）。
- **交互**：点击游戏条目，进入对应游戏的视频列表页。
- **视觉**：顶部导航栏使用辅助色 `#20B2AA`，列表项为白底黑字，清晰易读。

### 4. 个人中心页（我）

- **功能**：展示用户头像、昵称，提供功能入口。
- **入口**：
  - 我的收藏：查看收藏的视频片段。
  - 通用设置：进入个人信息设置页。
  - 退出登录：清除登录状态，返回登录页。

### 5. 设置页

- **功能**：管理个人信息，提供修改入口。
- **入口**：
  - 名字：跳转至更改名字页。
  - 手机号：跳转至修改手机号页。
  - 邮箱：跳转至修改邮箱页。
  - 密码：跳转至更改密码页。

### 6. 更改名字页

- **功能**：输入新昵称，点击「保存」完成更新。
- **交互**：保存成功后返回个人中心页，昵称同步更新。

### 7. 更改密码页

- **功能**：验证旧密码后，输入并确认新密码完成修改。
- **校验**：新密码与确认密码需一致，旧密码需验证通过。
- **交互**：修改成功后返回设置页，需重新登录生效。

### 8. 视频列表页

- **功能**：展示所选游戏下的所有视频条目。
- **交互**：点击视频条目，进入视频播放控制页。
- **视觉**：与游戏列表页保持一致的导航栏样式，保证体验连贯。

### 9. 视频播放控制页

- **功能**：提供视频播放控制功能。
- **控件**：
  - 播放/暂停按钮：切换播放状态。
  - 倍速调节：支持 0.5x、1.0x 等多档变速。
  - 重置进度：跳转到开头进行播放。
  - 循环播放：拖动进度条确定循环播放片段。
  - 关闭按钮：退出播放，返回视频列表页。

## 四、交互说明

1.  **页面跳转**
    - 登录页 → 注册页：点击「新用户注册」。
    - 登录/注册成功 → 游戏列表页（首页）。
    - 游戏列表页 → 视频列表页：点击游戏条目。
    - 视频列表页 → 播放控制页：点击视频条目。
    - 游戏列表页 → 个人中心页：点击用户头像。
    - 个人中心页 → 设置页：点击「通用设置」。
    - 设置页 → 更改名字/密码页：点击对应选项。
    - 所有页面左上角返回箭头 → 上一级页面。
    - 个人中心页 → 登录页：点击「退出登录」。

2.  **数据操作**
    - 更改名字/密码：输入完成后点击「保存」/「确认」提交，成功后返回上一页。
    - 视频播放：支持暂停、变速、音量调节、进度跳转等操作。

3.  **状态反馈**
    - 输入不合法/操作失败时，弹出 Toast 或对话框提示。
    - 加载状态显示 loading 动画，提升用户感知。
